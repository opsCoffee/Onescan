# Product Overview

OneScan is a professional BurpSuite extension for recursive directory scanning and vulnerability discovery. The plugin helps security researchers discover hidden APIs, sensitive information leaks, and test for unauthorized access by recursively scanning directory paths with configurable payloads.

## Core Capabilities

- **Recursive Directory Scanning**: Automatically scans all directory levels from discovered URLs
- **Fingerprint Recognition**: Identifies web technologies and frameworks using a rule-based matching system
- **Dynamic Variables**: 30+ variables for intelligent payload generation (domain, subdomain, timestamp, etc.)
- **Data Collection**: Automatically extracts and categorizes JSON fields and web names
- **Payload Processing**: Rule-based request manipulation (URL, headers, body modifications)
- **QPS Control**: Token bucket algorithm for rate limiting (1-9999 QPS)

## Target Users

Security researchers and penetration testers using BurpSuite Professional for web application security testing.

## Current Version

2.2.0
